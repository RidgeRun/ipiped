Description

Image Pipe Daemon (ipiped) is a daemon for controlling the video capture subsystem on Texas Instruments Davinci™ family of processors thru DBus interface. It includes an cli control tool.

Ipiped can be integrated with RidgeRun's AEW/AE library to provide this functionality to image sensors that doesn't implement it.
